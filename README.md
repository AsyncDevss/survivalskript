# 🚀 Survival Premium Sistemleri

Bu paket, bir Survival sunucusunun ihtiyaç duyduğu en kritik 20 mekaniği tek dosyada sunar. Giriş-çıkış mesajı veya sohbet engeli gibi gereksiz "spam" kodları içermez; doğrudan oyun oynanışına etki eden sistemler içerir.

## 📦 İçerik Özeti
- **Teleportasyon:** TPA ve Home sistemleri.
- **Ekonomi/Ticaret:** Eşya satma ve otomatik blok dönüştürme.
- **PvP & Ölüm:** Kafa düşürme, PvP aç/kapat ve Puan sistemi.
- **Kolaylık:** Repair, Çöp, Enderchest, Zırh Standı.
- **Doğa & Çevre:** Timber (Hızlı ağaç), Otomatik yer temizliği, Balıkçılık bonusu.

## 🛠️ Kurulum
1. `survival_premium.sk` dosyasını `plugins/Skript/scripts/` içine atın.
2. `/sk reload survival_premium` komutu ile başlatın.

## 🔑 Öne Çıkan Komutlar
| Komut | Açıklama |
| :--- | :--- |
| `/tpa /tpkabul` | Işınlanma sistemi. |
| `/sethome /home` | Ev sistemi. |
| `/sat` | Envanterdeki kömürleri paraya çevirir. |
| `/pvp` | Oyuncu PvP durumunu değiştirir. |
| `/repair` | Elindeki eşyayı tamir eder. |
